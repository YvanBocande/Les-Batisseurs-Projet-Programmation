II - Comment démarer le jeu

Il est nécessaire d'avoir java pour lancer le programme (lien pour le télécharger : https://www.java.com/fr/download/)

Sous Windows :

Ouvrir le cmd dans le dossier class: 


Pour lancer le jeu :

	- écrire "java -jar Les_Batisseurs.jar" et appuyer sur entrée. Vous pouvez maintenant jouer !
	- taper sur T ou G en fonction de la version que vous souhaitez lancer (textuelle ou graphique).


Sous Linux :

Dans l'explorateur de fichier, faites clique droit en vous assurant de bien être dans le dossier class qui contient Les_Batisseurs.jar, et cliquer sur "Ouvrir dans un terminal ici"

Pour la version terminal :

	- écrire "java -jar Les_Batisseurs.jar" et appuyer sur entrée. Vous pouvez maintenant jouer !
	- taper sur T ou G en fonction de la version que vous souhaitez lancer (textuelle ou graphique).
